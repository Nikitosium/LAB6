# RS-31_Haponko_Nikita_RTOS_LR6

**Номер групи:** РС-31

**Автор:** Гапонько Нікіта

**ОПЕРАЦІЙНІ СИСТЕМИ РЕАЛЬНОГО ЧАСУ ТА 32-БІТНІ МК**
**Лабораторна робота 6.** Технологія використання широтно-імпульсної модуляції (ШІМ) при розробці ПЗ ОСРЧ.

**Обладнання:** STM32F411 Discovery, CubeMX, Keil.

**Мета:** Ознайомитися з принципами генерації ШІМ-сигналу за допомогою апаратних таймерів STM32, навчитися керувати яскравістю світлодіода та змінювати параметри сигналу в динаміці.

__________________________________________________________________________________________

**Завдання до лабораторної роботи (Варіант 4):**
1. Реалізувати генерацію ШІМ-сигналу на виводі **PD15** за допомогою таймера **TIM5** з частотою **5 кГц**.
2. Реалізувати зміну коефіцієнта заповнення (duty cycle): кожне натискання кнопки **PA0** повинно збільшувати яскравість на **5%**.

Цей проєкт демонструє використання ШІМ для керування потужністю (яскравістю), що подається на периферійний пристрій, та обробку зовнішніх переривань для модифікації параметрів таймера.

### 👨🏼‍💻 Функціонал:
* **PWM Brightness Control**: Апаратне керування світлодіодом PD15 через канал таймера TIM5.
* **Dynamic Duty Cycle**: Програмна логіка, що змінює значення регістра порівняння (CCR) при кожній події від кнопки.
* **Frequency Stability**: Стабільна частота 5 кГц забезпечує відсутність видимого мерехтіння світлодіода.

### 📋 Ключові моменти реалізації:
1. **PWM Setup**: Налаштування таймера в режим PWM Generation CH4 з розрахунком періоду (ARR) для частоти 5 кГц.
2. **Duty Cycle Step**: Розрахунок кроку зміни значення порівняння, що відповідає 5% від повного періоду.
3. **Button Handling**: Обробка натискання кнопки з програмним антидребезгом для чіткого інкременту яскравості.
